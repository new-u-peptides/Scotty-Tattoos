# /public/tattoos — finished tattoo work

Drop finished tattoo photos here. These feed the portfolio grids on both
sites and the featured-work section on each home page.

## What we need

Aim for a **mix** — different styles, body placements, and zoom levels.
The portfolio grid uses portrait, square, and wide aspect ratios, so
variety in framing helps.

### Style coverage (covers both sites' filters)

- **Mandala / dotwork** — the studio's signature. 4–6 pieces, varied scale.
- **Geometric / sacred geometry** — Metatron, vesica piscis, flower of life.
- **Blackwork** — bold negative space, solid fills.
- **Fine line / single-needle** — delicate, restrained.
- **Ornamental** — lace, sternum, decorative borders.
- **Traditional / Japanese / colour** — for scottymassa.com's full style range.
- **Cover-up / rework** — before-and-after pairs are particularly compelling.

### Placement coverage

Mix body parts so the grid doesn't read as nine arm shots:
- Sternum / chest / collarbone
- Back / shoulder
- Arm / forearm / wrist
- Hand / fingers
- Hip / side / ribs
- Thigh / calf / shin / full leg
- Spine / neck

### Zoom levels

- **Wide** — full piece in context (body + environment).
- **Mid** — the piece filling most of the frame.
- **Tight** — close-up on detail / line quality / dot density.

A few of each gives the grid rhythm.

## File specs

- **JPG**, sRGB, ~1600 px on the long edge for the final web version
  (I'll resize; just send the best original you have).
- Background prefer studio / natural — avoid clutter behind the subject
  if possible.
- Healed photos read better than fresh-ink photos (less redness), but
  fresh ones add process variety. Mix both.
