# /public/studio — studio interior

Drop interior shots of the studio here. These feed the About / Studio /
Contact sections on both sites.

## What we need

Six to eight well-lit shots that show the space, ideally at the same time
of day so the colour temperature stays consistent across the site.

| Shot | Use across both sites |
|---|---|
| **Entrance / lounge** (wide) | About hero on both sites |
| **Reception desk** (with MASSA signage) | Contact page header on both sites |
| **Treatment room** — chair, lighting, equipment | About / Process step on both |
| **Consultation room** — desk, plants, the meeting space | About / Process step on both |
| **Flash wall** — framed art / pinup wall | About split-media block |
| **Tool detail** — ink pots, machines, sketchbook | Process step, blog headers |
| **Storefront / exterior** | Contact footer / map area |
| **Atmospheric mood shot** (low-light, no people) | Hero background option |

## Style notes

- **No people in frame** for the architectural shots. Empty room reads
  cleaner; we have separate portrait shots for people.
- **Natural light + room lights both on** — the warm wood + black-tile
  contrast is part of the studio's character; don't blow it out.
- **Wide angle** for the room shots (around 24–35 mm equivalent). The
  treatment room shot, especially, benefits from showing the whole setup.
- **Multiple angles** of the same room are fine — submit your favourites,
  I'll pick.

## File specs

- **JPG**, sRGB, ~1920 px on the long edge for the final web version.
- Landscape (3:2) preferred for hero / wide tiles; portrait (4:5) fine
  for split-media blocks.
- HDR / bracketed exposures fine — send the JPG, not the raw stack.
